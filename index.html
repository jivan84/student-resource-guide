This is a developer guide for students working on programming assignments and projects. Please note that it is not meant to be a normative guide for all CS classes (unless your instructor has explicitly pointed you to this guide).

In this guide, you will find the following:

  • Software Environment: A description of the software environment provided by the CS department, and which you will be expected to use to a certain extent.

  • Working Remotely: Suggestions and guidelines on how to access the CS software environment remotely.

  • Working in Teams: Suggestions for how to work effectively in pairs and in teams.
